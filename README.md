# Задача Стефана: таяние снежинки

2D моделирование фазового перехода лёд–вода методом конечных разностей с энтальпийной формулировкой.

## Метод

Решается уравнение теплопроводности с фазовым переходом:

$$\frac{\partial H}{\partial t} = k \, \nabla^2 T$$

Связь температуры и энтальпии:

$$T(H) = \begin{cases} T_{melt} + H / (\rho c_s), & H < 0 \quad \text{(лёд)} \\ T_{melt}, & 0 \le H \le \rho L \quad \text{(фазовый переход)} \\ T_{melt} + (H - \rho L)/(\rho c_l), & H > \rho L \quad \text{(вода)} \end{cases}$$

Энтальпийный метод позволяет обрабатывать границу плавления неявно — без отслеживания фронта.

## Результат

Снежинка (из PNG-маски) помещается в тёплую воду. Тонкие лучи тают первыми, центр — последним.

## Валидация

Положение фронта плавления в 1D сравнивается с аналитическим решением задачи Стефана:

$$s(t) = 2\lambda\sqrt{\alpha t}$$
