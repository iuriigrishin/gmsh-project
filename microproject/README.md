# 1 День
## Физическая модель и энтальпийный подход

Все расчёты основаны на энтальпийном методе решения задачи Стефана (фазовый переход лёд–вода). Вместо вспомогательной подвижной границы используются энтальпия $H$ и температура $T$, связанные через физические свойства материала.

---

### Основные определения

- Энтальпия $H$ — это **теплосодержание** системы при постоянном давлении. В коде она хранится в двумерной сетке `data_` и доступна через:
  ```cpp
  double& Grid::H(int i, int j);
  double Grid::H(int i, int j) const;
  ```
- Материал характеризуется:
  - плотностью $\rho$,
  - теплоёмкостями $c_{\text{solid}}$, $c_{\text{liquid}}$,
  - коэффициентами теплопроводности $k_{\text{solid}}$, $k_{\text{liquid}}$,
  - теплотой плавления $L$,
  - температурой плавления $T_{\text{melt}}$.

---

### Выбор нулевого уровня энтальпии

В коде энтальпия отсчитывается относительно температуры фазового перехода:
$$
H = 0 \quad \text{при } T = T_{\text{melt}}.
$$
Это позволяет:
- при $T < T_{\text{melt}}$ — хранить “отрицательную” энтальпию льда,
- при $T > T_{\text{melt}}$ — хранить “положительную” энтальпию воды,
- на интервале $0 \le H \le \rho L$ — описывать зону фазового перехода.

---

### Температура из энтальпии: `T(i, j)`

Функция  
```cpp
double Grid::T(int i, int j) const
```
переводит энтальпию $H$ в температуру $T$. Внутри используются три режима:

- Если $H < 0$, материал в твёрдой фазе ($T < T_{\text{melt}}$):
  $$
  T = T_{\text{melt}} + \frac{H}{\rho c_{\text{solid}}}.
  $$
- Если $H > \rho L$, материал в жидкой фазе ($T > T_{\text{melt}}$):
  $$
  T = T_{\text{melt}} + \frac{H - \rho L}{\rho c_{\text{liquid}}}.
  $$
- Если $0 \le H \le \rho L$, это зона фазового перехода (mushy) и температура фиксируется:
  $$
  T = T_{\text{melt}}.
  $$

Это отражает физику, где:
- до плавления энергия идёт только на охлаждение/нагрев льда,
- во время плавления всё тепло уходит в скрытую теплоту $L$,
- после плавления — в нагрев воды.

---

### Фаза по энтальпии: `phase(i, j)`

Функция  
```cpp
Phase Grid::phase(int i, int j) const
```
классифицирует состояние материала:

- $H < 0$ → `Phase::Solid` (лёд),
- $H > \rho L$ → `Phase::Liquid` (вода),
- $0 \le H \le \rho L$ → `Phase::Mushy` (смесь фаз).

Это стандартный энтальпийный критерий: фаза полностью определяется по значению $H$ относительно $\rho L$.

---

### Теплопроводность в зависимости от фазы: `conductivity(i, j)`

Функция  
```cpp
double Grid::conductivity(int i, int j) const
```
задаёт локальную теплопроводность:

- $H < 0$ → $k = k_{\text{solid}}$,
- $H > \rho L$ → $k = k_{\text{liquid}}$,
- в зоне перехода $0 \le H \le \rho L$ используется линейная интерполяция:
  $$
  k = (1 - \alpha) k_{\text{solid}} + \alpha k_{\text{liquid}}, \quad \alpha = \frac{H}{\rho L}.
  $$

Это численное приближение, позволяющее плавно переходить между свойствами твёрдой и жидкой фаз.

---

### Преобразования $H \leftrightarrow T$

Функции  
```cpp
double Grid::H_from_T_solid(double T_celsius) const;
double Grid::H_from_T_liquid(double T_celsius) const;
```
реализуют обратные формулы:

- для льда:
  $$
  H = \rho c_{\text{solid}}(T_{\text{celsius}} - T_{\text{melt}}),
  $$
- для воды:
  $$
  H = \rho L + \rho c_{\text{liquid}}(T_{\text{celsius}} - T_{\text{melt}}).
  $$

Они позволяют инициализировать температурное поле или визуализировать результат как температуру, при этом все вычисления происходят в терминах энтальпии.
